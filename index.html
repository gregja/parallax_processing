Parallax Processing

Based on a tutorial of Kris Hadlock (published in 2012)

Code upgraded with some of the best practices of 2019

(API Promise and Touch, function Path2D, strict mode, IIFE, etc.)

Plus an easing function (of Robert Penner) when clicking on the jeep (for fun)


You can use the arrow keys to move the jeep in the 4 directions.

You'll find a lot of comments into the Javascript code.
